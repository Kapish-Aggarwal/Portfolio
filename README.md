# Kapish Aggarwal — Portfolio

A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Designed with a dark editorial aesthetic featuring a 12-column grid layout, monospace typography, and smooth scroll-triggered animations.

---

## 🖥️ Live Preview

> Deploy to [Vercel](https://vercel.com) or [Netlify](https://netlify.com) and add your live URL here.

---

## 📁 Project Structure

```
portfolio/
├── index.html       # Markup and structure
├── style.css        # All styles, variables, animations, responsive
└── script.js        # Scroll animations, active nav, dynamic year
```

---

## ✨ Features

- **Dark grid aesthetic** — subtle CSS grid texture background with a 12-column card layout
- **Scroll-triggered animations** — cards fade up as they enter the viewport via `IntersectionObserver`
- **Active nav highlighting** — current section is highlighted in the navbar while scrolling
- **Fully responsive** — collapses gracefully from 12-column → 2-column → 1-column on mobile
